Providencie os Utensílios

Para ajudar evitar o desespero no meio de uma receita, providencie os utensílios básicos de cozinha para quem está dando os seus primeiros passos na mesma.

Lavar Louça

Se tem uma coisa que nos desencoraja a cozinhar é a quantidade de louça a ser lavada depois. Evite acumular louça de várias refeições, pois quanto mais cheia a pia maior será a preguiça.

Siga a Receita

Receitas existem por um motivo e, no começo ,os improvisos costumam dar errado. Deixe está parte para quando tiver mais experiência.

Chame os Amigos

Esta é a principal motivação para se conzinhar. É ótimo receber pessoas para comer algo que você preparou especialmente para elas.
